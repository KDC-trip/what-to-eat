# 🍱 今天吃什麼？- 隨機美食決定器 (What To Eat)

![License](https://shields.io)
![Language](https://shields.io)
![Status](https://shields.io)

**「今天吃什麼」** 是一個專為解決選餐困難而設計的實用工具。透過趣味的大富翁與拉霸機模式，幫你快速從口袋名單中選出下一餐！

🔗 **[👉 立即點擊使用](https://github.io)**

---

## 🌟 核心功能

*   **🕹️ 雙遊戲模式**：內建「正餐大富翁」與「夜市拉霸機」，讓決定美食變有趣。
*   **📍 Google 地圖導航**：抽到結果後可一鍵跳轉至 Google Maps 搜尋附近餐廳。
*   **💾 自動記憶功能**：你的自定義美食清單會自動儲存在瀏覽器，下次開啟時無需重新輸入。
*   **📱 手機優化設計**：介面簡潔、載入快速，適合出門在外單手操作。

---

## 🚀 如何使用

1. **選擇模式**：切換正餐、點心或夜市模式。
2. **自定義清單**：在輸入框輸入餐廳名稱並點擊「新增」，或按 ✕ 刪除。
3. **開始決定**：點擊「選一個」或「拉霸手把」。
4. **前往用餐**：點擊結果下方的「📍 點我查看地圖導航」即可出發！
