# 🍱 美食全攻略 - 今天吃什麼？ (What To Eat)

![License](https://shields.io)
![Language](https://shields.io)
![Author](https://shields.io)

**「美食全攻略」** 是一個專為外食族設計的隨機美食決定工具。透過趣味的遊戲化介面（大富翁、拉霸機），解決你每天「今天吃什麼」的世紀難題！

🔗 **[👉 立即點擊體驗網頁版](https://kdc-trip.github.io/what-to-eat/)**

---

## 🌟 核心功能

*   **🕹️ 三大趣味模式**：
    *   **正餐大富翁**：用走格子的方式選出你的午晚餐。
    *   **點心驚喜盒**：下午茶時間不知道吃什麼？點一下就知道。
    *   **夜市拉霸機**：模擬夜市歡樂氣氛，一拉即定。
*   **✏️ 自定義菜單**：可以隨時新增或刪除你口袋名單中的餐廳。
*   **📱 響應式設計**：專為手機優化，單手即可操作，出門在外隨時決定。
*   **⚡ 極速載入**：純原生 JavaScript 開發，不佔手機空間，秒開即玩。

## 🚀 如何使用

1. **選擇模式**：在網頁上方切換「正餐」、「點心」或「夜市」。
2. **調整清單**：在輸入框輸入你想吃的店家名稱並點擊「新增」，或是按 ✕ 刪除。
3. **開始決定**：點擊「選一個」或「拉霸手把」。
4. **結果揭曉**：系統會自動為你挑選出今天的命定美食！
